# 学生管理系统 java452

## 项目概述

学生管理系统是一款基于 Java、SpringBoot 和 Vue 技术栈实现的前后端分离项目。该项目拥有14张表关联设计，功能全面，覆盖了系统管理、教学管理、学生管理、教师管理等多个业务模块。
